# Software Engineering Course

## Prerequisites
* Allocate Ubuntu VM (18.04)
* Install helper - [talan cli](https://github.com/project-talan/tln-cli)
* Install docker & docker compose
  ```
  tln install docker:docker-compose
  ```

## Configure workspace 
* Create project's home
  ```
  cd ~
  mkdir projects
  cd projects
  tln init-conf
  ```
